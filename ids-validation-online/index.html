FAQ

What is IDS?
Information Delivery Specification — a buildingSMART open standard for defining and automatically checking the information requirements of an IFC model.
Which IDS version do you support?
IDS 1.0 — the current buildingSMART release.
Can I check data quality without a full IDS?
Yes. ClashControl ships with a built-in data quality checker covering BIM basics (missing GlobalIds, generic names, missing materials, unassigned storeys, empty geometry), an ILS pack, and NL-SfB classification checks — no IDS file required.
