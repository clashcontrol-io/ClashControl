FAQ

Is there a file size limit?
No hard limit. Practical model size depends on your browser's memory — typical workstation browsers comfortably handle multi-hundred-megabyte IFCs. Larger or federated models can be parsed in the optional localhost engine.
Does my model upload to a server?
No. IFC parsing and viewing run fully client-side in your browser. Nothing is uploaded.
What else can it do beyond viewing?
Hard and soft clash detection, BCF 2.1 / 3.0 import and export, IDS validation, data quality checks, and a live link to Revit for two-way sync.
