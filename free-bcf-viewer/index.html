FAQ

What is BCF?
BIM Collaboration Format — an open standard for exchanging model issues (a comment + a 3D viewpoint + the affected elements) between different tools.
Which tools can open my BCF?
Any BCF-compatible tool: Solibri, BIMcollab, Revit, Navisworks, and more.
Does it round-trip?
Yes — ClashControl imports BCF as well as exporting it, so issues raised elsewhere can be loaded, reviewed, and re-exported. BCF 2.1 and 3.0 are both supported.
