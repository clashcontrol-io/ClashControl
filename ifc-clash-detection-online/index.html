FAQ

What's the difference between hard and soft clashes?
A hard clash is geometry physically overlapping. A soft clash is a clearance/tolerance violation — things too close even if they don't touch. ClashControl checks both.
Do my models get uploaded anywhere?
No — IFC parsing and clash detection run fully client-side in your browser. An optional localhost engine can be enabled for exact-mesh boolean ops, also without leaving your network.
Which IFC versions are supported?
IFC 2x3 and IFC4, via the web-ifc WASM library.
